Thank your for using the Sovren Product Demo for Bimetric Scoring!

This folder contains the following files:
 - BimetricScoringRespose.json (The full response from the Bimetric Scorer API call as JSON)
 - BimetricScoringRespose.xml (The full response from the Bimetric Scorer API call as XML)

Additionally, a folder has been created for the Source Document and Target Documents. For each of those files, a folder with that file name has been created. Inside each file folder, you will find the following documents:
 - [FileName].json (The standard parsed document as JSON)
 - [FileName].xml (The standard parsed document as XML)
 - [FileName].html (The HTML representation of the document text. We also support outputting the document as PDF and RTF with a simple setting in the API request)
 - [FileName] (The original file that was uploaded)
